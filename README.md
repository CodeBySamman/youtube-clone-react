
# VidTube React YouTube Clone

A responsive YouTube-inspired web application built with React and Tailwind CSS.

## Features

- Browse Popular Videos
- Watch Videos
- View Channel Details
- Display Comments
- Category-Based Videos
- Responsive Design

## Technologies Used

- React
- React Router
- Tailwind CSS
- Express.js
- YouTube Data API
- Moment.js
